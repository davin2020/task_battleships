Task Battleships

Part 1 - Add New Tasks

Add the tasks that you want to appear in your Batthship Grid.

If you enter less than 15 tasks, "empty" tasks will be added, which can represent your main or ongoing project.

When you have finished adding tasks, goto Part 2 to load the tasks into the grid.

  1. existing tasks will appear here

Part 2 - Load Task Battleship Grid

Click the "Shuffle and Load" button to load the above tasks into the battleship grid - once loaded, you can't add more tasks.

Then click each ? square to reveal the hidden task, which has an orange background to indicate "Work in Progress".

Once you have completed a task, click it again to change the backgroun colour to green to indicate a "Completed" task - Congrats!

CAUTION - If you refresh the website or click the "Shuffle" button again, you will lose the current status of your tasks (ie orange or green) and have to start over.


Once you have done all the tasks, Refresh the website to start again!
